# Accessibility Auditor

Scan web app code for WCAG 2.1 violations — color contrast, alt text, ARIA labels, keyboard navigation, and semantic HTML.

## Install

### Global (all projects)

```bash
mkdir -p ~/.claude/skills
cp SKILL.md ~/.claude/skills/accessibility-auditor.md
```

### Project-level (single project)

```bash
mkdir -p .claude/skills
cp SKILL.md .claude/skills/accessibility-auditor.md
```

Claude Code will automatically detect and use the skill when relevant.
