# API Docs Writer

Generate comprehensive API documentation from route handlers with endpoints, parameters, request/response examples.

## Install

Copy the `SKILL.md` file to your Claude Code skills directory:

```bash
# Global install
cp SKILL.md ~/.claude/skills/api-docs-writer.md

# Project-level install
mkdir -p .claude/skills
cp SKILL.md .claude/skills/api-docs-writer.md
```

## Usage

Once installed, Claude Code will automatically use this skill when you ask it to document APIs, create API docs, or generate endpoint documentation.
