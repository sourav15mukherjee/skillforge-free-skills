# Changelog Generator

Auto-generate a polished CHANGELOG.md from your git history — grouped by type, linked to commits, ready to ship.

## What it does

- Reads git commits since the last tag (or any range)
- Groups by type: Added, Fixed, Changed, Security, Breaking Changes
- Follows Keep a Changelog format
- Suggests semantic version bumps
- Links to commit hashes

## Usage

```
> Generate a changelog for this release
```

The skill will analyze your commits and produce a formatted entry ready to paste into CHANGELOG.md.
