# Commit Message Generator

Generate conventional commit messages from staged git changes following the Angular/Conventional Commits standard.

## Install

Copy the `SKILL.md` file to your Claude Code skills directory:

```bash
# Global install
cp SKILL.md ~/.claude/skills/commit-message-generator.md

# Project-level install
mkdir -p .claude/skills
cp SKILL.md .claude/skills/commit-message-generator.md
```

## Usage

Once installed, Claude Code will automatically use this skill when you ask it to commit changes, write a commit message, or help with git commits.
