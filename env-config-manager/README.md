# Environment Config Manager

Manage .env files, validate required variables, generate .env.example templates, and detect missing config between environments.

## Install

### Global (all projects)

```bash
mkdir -p ~/.claude/skills
cp SKILL.md ~/.claude/skills/env-config-manager.md
```

### Project-level (single project)

```bash
mkdir -p .claude/skills
cp SKILL.md .claude/skills/env-config-manager.md
```

Claude Code will automatically detect and use the skill when relevant.
