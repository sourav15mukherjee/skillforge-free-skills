# Error Explainer

Paste any error message or stack trace — get a plain-English explanation, root cause analysis, and a working fix.

## What it does

- Parses error messages, stack traces, and build failures
- Reads your actual source code to understand context
- Explains what went wrong in plain English
- Provides exact before/after code fixes
- Suggests how to prevent the error in the future

## Supported errors

- JavaScript/TypeScript runtime errors
- React hydration mismatches
- Next.js build failures
- Node.js module resolution errors
- Type errors and compilation failures
- And more

## Usage

Install the skill, then paste any error into Claude Code:

```
TypeError: Cannot read properties of undefined (reading 'map')
    at UserList (src/components/UserList.tsx:14:22)
```

The skill will read your source file, identify the root cause, and suggest a fix.
