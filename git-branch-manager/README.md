# Git Branch Manager

Create, manage, and clean up git branches following team naming conventions with smart suggestions.

## Install

Copy the `SKILL.md` file to your Claude Code skills directory:

```bash
# Global install
cp SKILL.md ~/.claude/skills/git-branch-manager.md

# Project-level install
mkdir -p .claude/skills
cp SKILL.md .claude/skills/git-branch-manager.md
```

## Usage

Once installed, Claude Code will automatically use this skill when you ask it to create a branch, switch branches, clean up branches, or manage git workflow.
