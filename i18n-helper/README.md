# i18n/Localization Helper

Extract hardcoded strings into translation keys, generate locale files, find untranslated strings, and validate locale completeness.

## Install

### Global (all projects)

```bash
mkdir -p ~/.claude/skills
cp SKILL.md ~/.claude/skills/i18n-helper.md
```

### Project-level (single project)

```bash
mkdir -p .claude/skills
cp SKILL.md .claude/skills/i18n-helper.md
```

Claude Code will automatically detect and use the skill when relevant.
