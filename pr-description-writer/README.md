# PR Description Writer

Generate comprehensive pull request descriptions from branch diffs with summary, changes, and test plan.

## Install

Copy the `SKILL.md` file to your Claude Code skills directory:

```bash
# Global install
cp SKILL.md ~/.claude/skills/pr-description-writer.md

# Project-level install
mkdir -p .claude/skills
cp SKILL.md .claude/skills/pr-description-writer.md
```

## Usage

Once installed, Claude Code will automatically use this skill when you ask it to create a PR, write a PR description, or prepare a pull request.
