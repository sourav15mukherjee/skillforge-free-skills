# Prompt Engineer

Turn vague instructions into optimized AI prompts — with system messages, few-shot examples, and structured output formats.

## What it does

- Crafts production-grade prompts for any LLM (Claude, GPT-4, Llama, etc.)
- Generates system messages with role, context, and constraints
- Creates few-shot examples for consistent output
- Defines structured output formats (JSON, markdown, etc.)
- Optimizes for the specific target model

## Usage

```
> Help me write a prompt for a code review bot
```

The skill will ask about your use case, target model, and constraints, then generate a complete prompt package.
