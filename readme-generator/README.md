# README Generator

Analyze project structure and generate a comprehensive README.md with badges, install instructions, usage examples, and API docs.

## Install

### Global (all projects)

```bash
mkdir -p ~/.claude/skills
cp SKILL.md ~/.claude/skills/readme-generator.md
```

### Project-level (single project)

```bash
mkdir -p .claude/skills
cp SKILL.md .claude/skills/readme-generator.md
```

Claude Code will automatically detect and use the skill when relevant.
