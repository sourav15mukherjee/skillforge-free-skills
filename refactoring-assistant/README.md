# Refactoring Assistant

Identify code smells and apply systematic refactoring patterns to improve code quality without changing behavior.

## Install

Copy the `SKILL.md` file to your Claude Code skills directory:

```bash
# Global install
cp SKILL.md ~/.claude/skills/refactoring-assistant.md

# Project-level install
mkdir -p .claude/skills
cp SKILL.md .claude/skills/refactoring-assistant.md
```

## Usage

Once installed, Claude Code will automatically use this skill when you ask it to refactor code, clean up code, improve code quality, or fix code smells.
