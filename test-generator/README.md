# Test Generator

Generate comprehensive unit and integration tests for any function, class, or module with edge cases and mocking.

## Install

Copy the `SKILL.md` file to your Claude Code skills directory:

```bash
# Global install
cp SKILL.md ~/.claude/skills/test-generator.md

# Project-level install
mkdir -p .claude/skills
cp SKILL.md .claude/skills/test-generator.md
```

## Usage

Once installed, Claude Code will automatically use this skill when you ask it to write tests, add tests, create test coverage, or test a specific function.
